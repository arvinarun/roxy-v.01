# 🖤 Life & Mood Assistant 🖤
A mini interactive Python assistant that checks your mood, energy, and available time, then gives personalized workout or self-care suggestions.

*"Your daily vibe matters… let’s make the most of it."* ⚡

---

## Features
- Interactive terminal-based Python program
- Mood, energy, and time inputs influence suggestions
- Personalized workout or self-care guidance based on your state
- Beginner-friendly, easy to read and expand
- Fun emojis to make the experience lively and engaging
